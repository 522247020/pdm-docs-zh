# pdm-docs-zh

PDM 中文文档（社区维护）

[📖 查看文档](https://pdm-project.org/zh-cn/)

## 维护者

- [mhwy](https://github.com/522247020)

## 贡献

在 `docs/` 目录下找到对应的文档文件，修改后提交 PR 即可。

安装依赖：

```
pdm install
```

预览文档：

```
pdm doc
```
